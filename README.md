## Build Setup
```
git clone https://github.com/jianfengtheboy/Music.git //下载Music

cd Music //进入Music目录

npm install (或者cnpm install) //安装依赖

npm run dev (或者cnpm run dev) //服务端运行

npm run build //项目打包
